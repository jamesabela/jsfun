🎤 The Math Challenge

Variables become useful when you can calculate with them. We know Taylor Swift was born in 1989. Let's write a program that asks for your birth year and computes the age difference!

💡 No tests in this exercise. Run the code and type in your birth year to see the result!

⚙️ how it works
TaylorSwiftBorn = 1989
MyAge = int(input('When were you born?'))
AgeDifference = MyAge - TaylorSwiftBorn
print("Taylor Swift is", AgeDifference, "years older than you")
💡 Notice how we use int() to tell Python that the input is a **number**, not text. Without it, the maths won't work!
🔢 AgeDifference = MyAge - TaylorSwiftBorn does the subtraction and stores the result in a new variable.

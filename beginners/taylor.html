LEVEL 9: VARIABLES & CALCULATIONS

Aging Fun 🎂

🎤 let us play!

Variables only really become useful when you can play with them. For example, you might want to know how much older Taylor Swift is than you. We know Taylor Swift was born in 1989. (Yes, she's really OLD!) Now all we need to know is when you were born and do the calculation.

💡 No tests in this exercise. Run the code and type in your birth year to see the result!
⚙️ how it works
TaylorSwiftBorn = 1989
MyAge = int(input('When were you born?'))
AgeDifference = MyAge - TaylorSwiftBorn
print("Taylor Swift is",AgeDifference,"years older than you")

Example run:

When were you born? 2010 Taylor Swift is 21 years older than you
💡 Notice how we use int() to tell Python that the input is a number, not text. Without it, the maths won't work!
🔢 AgeDifference = MyAge - TaylorSwiftBorn does the calculation and stores the result in a new variable.

Core Task Write a Joke
Write a program that outputs a joke. The joke and the punchline should output on separate lines.
💡 example

Here's an example of how your program might look:

print("Why did the chicken cross the road?")
print("To get to the other side!")

Example output:

Why did the chicken cross the road? To get to the other side!
💡 Use a separate print() statement for each line of the joke.
🎯 Be creative! Find or make up a joke that will make your classmates laugh.

Core Task Write a Joke
Write a program that outputs a joke. The joke and the punchline should output on separate lines.
Extension Comic Timing ⏱️
You can use the time module to help give your joke some comic timing! Make the audience wait for the punchline.
⏱️ time module example

Python's time module lets you pause the program. Use time.sleep(seconds) to add a dramatic pause before the punchline:

import time
print("Wait for it")
time.sleep(2)
print("Worth waiting for!")

Example output:

Wait for it ... 2 second pause ... Worth waiting for!
💡 import time goes at the top of your program.
🎯 time.sleep(2) pauses for 2 seconds. Try different numbers for different effects!

💡 What is a Variable?

Think of a variable as a labeled **storage box** in your computer's memory. You put a value (like text or a number) inside, and give the box a name so you can find it later.

📦 Interactive Variable Playground:

firstName "James"
lastName "Bond"
print(firstName, lastName)
James Bond
🔍 Your Coding Mission

In Python, you can print multiple variables on the same line by separating them with commas. You can also print the same variable more than once if you need to!

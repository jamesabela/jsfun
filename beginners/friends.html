🔍 your mission
💡 The Challenge: How can you print all 4 names using variables? What do you need to add?
🎯 Objective: Make the code print the names Ross Monica Phoebe Joey at the end.
⚙️ code guidance

When a variable is assigned a new value, the old value is overwritten and lost:

Starter code:

name1 = "Ross"
name2 = "Monica"
name3 = "Joey"

print(name3) # Prints Joey

name3 = "Phoebe"

print(name3) # Prints Phoebe

# How can you print all 4 names using variables?
print(name1, name2, name3)

Expected Final Output line:

Ross Monica Phoebe Joey

Solution: Store the value of name3 in a new variable (like name4 = name3) before name3 is updated to "Phoebe". Then, print all four variables at the end!

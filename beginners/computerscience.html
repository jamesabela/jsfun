💡 The Challenge: Learn how to start a new line within a single print statement.
🎯 Objective: Output the words I, love, and Computer Science on three separate lines.
⚙️ code guidance

In Python, putting \n inside a string tells the console to start a new line immediately. For example:

print("Hello\nWorld")

Outputs:

Hello World

Starter code:

print("I got nothing to say")

Expected Output:

I love Computer Science

Replace the starter text inside the print statement with the three words separated by \n to complete the mission!

👋 Introduction to Input

Now we can input our own things! The input() function lets your program ask a question and wait for the user to type an answer.

💡 No tests in this exercise. Don't forget to type in your name when you run it!
🕹️ Interactive Greeting Simulator

Type a name and choose a secret agent persona to see how Python variables dynamically construct a print message!

name = input("Hi! What's your name?")
Hi! What's your name? Alice
print("Hi", name)
Hi Alice
⚙️ how it works

Look at the code carefully:

name = input("Hi! What's your name? ")
print("Hi", name)
💡 input() shows the message in brackets and waits for you to type something.
🎯 Whatever you type gets stored in the variable name, which is then used in the print().
🌍 extension: try a different language!

Why not try greeting someone in a different language? Change the message inside input() and print() to say hello in French, Spanish, or any language you like!

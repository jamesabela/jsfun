# Python Beginners Puzzle Course

This course is designed for Python Code Lab.

All Python files and tutorial HTML files are in one folder:

`beginners`

Each Python file starts as a mini puzzle. The starter code is intentionally incomplete or incorrect, so students need to edit the code to pass the tests.

## Includes

1. **speechless.py** - Print a quote from Aladdin: "All I know is I won't go speechless".
2. **computerscience.py** - Print a simple string: "I, love, Computer Science".
3. **jokes.py** - Output a joke and a punchline on separate lines.
4. **variables.py** - Fix a print statement to output the spy catchphrase: "Bond James Bond".
5. **maths.py** - Use basic math operations to print the number 15.
6. **friends.py** - Use variables to print a sequence of names: "Ross Monica Phoebe Joey".
7. **assignment.py** - Assign name and favorite food to separate variables and output them.
8. **hello.py** - Simple input/output greeting a user by name.
9. **taylor.py** - Perform calculations comparing the user's birth year with Taylor Swift's birth year.
10. **celebchat.py** - Interactive celebrity chatbot simulating a conversation.

# Syllabus Notes: A-level 9618 Data Structures

This course is suitable for introducing the A-level 9618 data structures topic before moving to OOP implementations.

## Covered

- Abstract Data Type (ADT)
- Stack using a Python list
- Push, pop and peek ideas
- Stack pointer idea
- Overflow and underflow
- Queue using a Python list
- Enqueue and dequeue
- Front pointer and rear pointer
- Circular queue wraparound
- Linked list using parallel Python lists
- Start pointer
- Next pointer
- Null pointer using `-1`
- Traversal, insertion and search

## Not covered yet

- OOP implementations using classes
- Binary trees
- Hash tables
- Graphs

These can be taught in later courses.

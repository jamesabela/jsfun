# Data Structures with Python Lists

This course introduces A-level 9618 data structures using normal Python lists. It avoids OOP because classes will be taught later.

## Purpose

Students learn how stacks, queues and linked lists work as abstract data types. The course includes animated HTML guides, Python starter puzzles, automated tests where useful, and a separate vocabulary reference page.

## Includes

1. `01_adt_intro.py` and `01_adt_intro.html` - ADT, stack, queue and linked list overview.
2. `02_stack_push.py` and `02_stack_push.html` - stack push and pop.
3. `03_stack_limits.py` and `03_stack_limits.html` - stack overflow and underflow.
4. `04_queue_basic.py` and `04_queue_basic.html` - enqueue and dequeue using a list.
5. `05_queue_pointers.py` and `05_queue_pointers.html` - front and rear pointers.
6. `06_circular_queue.py` and `06_circular_queue.html` - circular queue wraparound with modulo.
7. `07_linked_list_traverse.py` and `07_linked_list_traverse.html` - linked list traversal using data and pointer lists.
8. `08_linked_list_insert.py` and `08_linked_list_insert.html` - inserting at the front of a linked list.
9. `09_linked_list_search.py` and `09_linked_list_search.html` - searching a linked list.
10. `10_final_project.py` and `10_final_project.html` - final creative project combining all three ADTs.
11. `vocabulary.html` - vocabulary page linked from every guide.
12. `teachers.md` - teacher notes and solution code.

## Breadcrumbs

Each HTML guide and Python file includes a progress marker such as `Level 4 of 10 - Data Structures`. These are progress breadcrumbs, not section navigation links.

## Animation Notes

The HTML guide files include embedded JavaScript animations. Students can step through stack push/pop, queue enqueue/dequeue and linked list pointer changes visually before editing the Python code.

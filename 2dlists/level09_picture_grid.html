Picture Grid

Turn a grid into visual output

What this means

Images on a screen are built from rows and columns of pixels. A simple 2D list can model this idea. Each letter can represent one colour or symbol.

A 2D list is a list of lists. The first index usually chooses the row. The second index usually chooses the column.
grid[row][column]
Real-life examples
  • Pixel art in games
  • LED display panels
  • Minecraft-style block maps
  • Tiny icons made from symbols

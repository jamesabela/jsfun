Update a Cell

Change one value inside the grid

What this means

2D lists are not just for reading data. You can also change one cell. This is useful when a seat is booked, a game character moves, or a spreadsheet value is corrected.

A 2D list is a list of lists. The first index usually chooses the row. The second index usually chooses the column.
grid[row][column]
Real-life examples
  • Marking a cinema seat as booked
  • Moving a player to a new map location
  • Changing a score in a table
  • Updating a classroom register

Level 6 of 10 - 2D Lists

Total a Grid

Add values stored in rows and columns

What this means

A 2D list can store numbers in a table. You may need to total each row, each column, or the whole table. This is useful for sales, scores, temperatures and survey results.

A 2D list is a list of lists. The first index usually chooses the row. The second index usually chooses the column.
grid[row][column]
Real-life examples
  • Total sales for each shop branch
  • Total marks for each student
  • Total steps recorded across several days
  • Total goals scored by several teams

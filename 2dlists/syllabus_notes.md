# Syllabus and Teaching Notes - 2D Lists

2D lists are useful for teaching tables, grids and repeated structures. In Cambridge-style Python work, they often appear as arrays with two indexes. Students should be comfortable with the idea that the first index selects the row and the second index selects the column.

Common exam-style tasks include:

- creating a fixed 2D list
- reading one item using two indexes
- updating one item using two indexes
- using nested loops to traverse all values
- searching for a value and reporting its row and column
- calculating totals from rows or columns
- validating row and column input before accessing a grid

The final project is deliberately open-ended because it gives students a reason to combine access, update, traversal and search in one creative program.

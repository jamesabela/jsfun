Level 7 of 10 - 2D Lists

Search a Grid

Find the row and column of a target

What this means

Searching a 2D list means checking each cell until the target is found. The program usually needs nested loops because it must check rows and columns.

A 2D list is a list of lists. The first index usually chooses the row. The second index usually chooses the column.
grid[row][column]
Real-life examples
  • Finding treasure on a game map
  • Finding a student name in a table
  • Finding an available room in a school timetable
  • Finding a product code in a warehouse grid

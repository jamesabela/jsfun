Level 10 of 10 - 2D Lists

Final Project

Build a small grid-based game

What this means

This project brings the course together. A grid-based game can store the world as a 2D list. The player chooses a row and column, and the program reveals what is stored there.

A 2D list is a list of lists. The first index usually chooses the row. The second index usually chooses the column.
grid[row][column]
Real-life examples
  • Treasure hunt games
  • Battleships-style guessing games
  • Escape room maps
  • Quiz boards where each square hides a question

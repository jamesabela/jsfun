What this means

A 2D list is useful when data has rows and columns. In Python, the first square bracket normally chooses the row and the second square bracket chooses the column. This is why array2d[2][3] means row 2, column 3, not x then y.

A 2D list is a list of lists. The first index usually chooses the row. The second index usually chooses the column.
grid[row][column]
Real-life examples
  • A spreadsheet cell in a table of marks
  • A cinema seat plan with rows and seats
  • A game map where the player chooses a location
  • A bus timetable with rows for routes and columns for times

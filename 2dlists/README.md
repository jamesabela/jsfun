# Python 2D Lists Course

This folder contains a Python Code Lab course on 2D lists. It introduces row and column indexing, nested lists, cell updates, nested loops, searching, totals, booking grids, picture grids and a final grid-based project.

## Breadcrumb Progress

Each tutorial page includes a progress breadcrumb such as `Level 1 of 10 - 2D Lists`. This is a progress marker only. It is not intended as navigation between sections.

## Includes

1. `level01_find_location.py` and `.html` - interactively find the row and column of a value.
2. `level02_create_grid.py` and `.html` - create a 3 by 4 2D list.
3. `level03_access_cells.py` and `.html` - access rows and individual cells.
4. `level04_update_cells.py` and `.html` - update a cell using user input.
5. `level05_nested_loops.py` and `.html` - use nested loops to visit every cell.
6. `level06_grid_totals.py` and `.html` - calculate row totals and a grand total.
7. `level07_search_grid.py` and `.html` - search a 2D list for a target.
8. `level08_booking_grid.py` and `.html` - create a simple seat booking system.
9. `level09_picture_grid.py` and `.html` - create visual output from a picture grid.
10. `level10_final_project.py` and `.html` - build a small grid-based project.

## Notes

The early levels use automated tests where there is a clear expected answer. The final project is intentionally more open-ended, so the test only checks that the starter project runs before students develop it further.

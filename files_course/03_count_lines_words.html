Count Lines and Words

Use loops to analyse a text file.

What this is about

A file can be processed one line at a time. This is useful because you do not need to know how long the file is before you start. A loop can count lines, search for words, or build statistics.

  • A search engine counts matching words.
  • A teacher tool counts answers in a class file.
  • A poetry tool can count how often a phrase appears.

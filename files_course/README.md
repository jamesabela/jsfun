# Files Course - Loading, Saving and Appending Files

This course teaches file handling in Python using Python Code Lab. It covers loading files with `#load`, reading text files, using filename variables, counting text, writing new files, appending to files, reading TSV data, searching records, saving filtered results and a final creative monster project.

## Includes

1. `01_load_poem.py` - Level 1 of 10 - load `rudyard.txt` using `#load` and read the first line.
2. `02_filename_variable.py` - Level 2 of 10 - use a variable to store the filename.
3. `03_count_lines_words.py` - Level 3 of 10 - loop through a file and count lines and words.
4. `04_write_stats.py` - Level 4 of 10 - write a new output file using `"w"` mode.
5. `05_append_log.py` - Level 5 of 10 - append to a log file using `"a"` mode.
6. `06_read_tsv.py` - Level 6 of 10 - read tab-separated values from `historical_events.tsv`.
7. `07_search_tsv.py` - Level 7 of 10 - search a TSV file by year.
8. `08_save_timeline.py` - Level 8 of 10 - filter TSV data and save it to a new file.
9. `09_append_tsv.py` - Level 9 of 10 - append a new structured row to a TSV file.
10. `10_monster_codex.py` - Level 10 of 10 - creative RPG monster-file project with no direct automated tests.

## Breadcrumbs

Each Python file and HTML page includes a progress breadcrumb, such as `Level 4 of 10 - Files Course`. This is not a navigation system. It simply tells students how far they are through the course.

## Data files

- `rudyard.txt` - text file used for reading and counting.
- `historical_events.tsv` - tab-separated data file used for structured file handling.
- The final level uses an external `#load` link for `Monsters.txt`.

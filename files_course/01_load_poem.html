Load a Text File

Use #load to bring a file into Python Code Lab.

Exam Pseudocode & Guide
What this is about

A normal Python program can open files that are stored beside it. In Python Code Lab, the browser needs to fetch the file first. The special #load line downloads the file into the virtual Python space before your code starts. After that, standard Python file handling works normally.

  • A poem reader can load rudyard.txt and print each line.
  • A quiz can load questions from a text file.
  • A game can load a list of monsters or items.

# Source Notes

This course was built from the supplied file-handling guide, the `rudyard.txt` poem file, and `historical_events.tsv`.

Key course ideas:

- `#load URL` can preload a file into Python Code Lab's virtual file space.
- `with open(filename, "r") as file:` is used for safe reading.
- `"w"` write mode creates a fresh file or replaces an existing file.
- `"a"` append mode adds to the end of a file without deleting previous content.
- `
` is needed when writing lines because `write()` does not automatically add a new line.
- TSV files use `	` between fields and are useful because they are simple, spreadsheet-friendly text files.
- The final creative project uses the Monsters.txt file from the Python RPG tutorial.

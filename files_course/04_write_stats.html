Write a New File

Use write mode to create a fresh output file.

Exam Pseudocode & Guide
What this is about

Write mode, "w", creates a new file or replaces an old one. It is useful when the output should be fresh each time the program runs.

  • A report generator writes a new summary file.
  • A quiz program writes final results.
  • A poem analyser writes statistics.

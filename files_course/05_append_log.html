Append to a File

Add to the end without deleting old data.

What this is about

Append mode, "a", keeps the existing file and adds new text at the bottom. It is useful for logs because every run can add a new line.

  • A login system appends sign-in records.
  • A game appends battle notes.
  • A reading tracker appends each reader.

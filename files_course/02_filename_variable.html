Filename Variables

Make file code easier to edit.

What this is about

A filename is just text. Storing it in a variable makes your program easier to adapt because you only change the filename in one place.

  • A teacher tool can switch between class1.txt and class2.txt.
  • A game can load monsters.txt or items.txt depending on the level.
  • A data project can store the source filename at the top of the program.

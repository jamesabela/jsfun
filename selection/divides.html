➗ Divisibility

Can we determine if a number divides exactly by 3? We can use selection alongside a special mathematical operator!

💡 The Modulo Operator (%): The % operator calculates the remainder of a division. For example, 10 % 3 is 1 (because 3 goes into 10 three times with 1 left over). If a number divides exactly, its remainder is 0 (e.g. num % 3 == 0).
🎯 Your Task

Ask the user for a number and output whether it is divisible exactly by 3:

  1. Read the number as an integer.
  2. Check if the number divides exactly by 3 (using % 3 == 0).
  3. If it divides exactly, output: 
    "That divides by 3"
  4. Otherwise, output: 
    "That doesn't divide by 3"

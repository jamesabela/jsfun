Seven Ate Nine 🦖

💡 Why was 6 afraid of 7?

Because 7, 8, 9! In Python, we can run multiple independent comparison checks using multiple if blocks. Unlike elif or else, which only run if the previous checks fail, separate if statements will all run if their conditions are met.

🎯 Your Task

Modify the code so that num is read as an integer from user input:

num = int(input("Enter a number: "))

Then, write three independent if statements to check the following:

  1. If num is greater than 5, print: bigger than 5
  2. If num is less than 10, print: smaller than 10
  3. If num is equal to 6, print: afraid
⚙️ How Tests Work: The test runner checks three numbers:
• Inputting 6 should trigger all checks and print bigger than 5, smaller than 10, and afraid.
• Inputting 3 should only print smaller than 10.
• Inputting 12 should only print bigger than 5.
📋 Original Code

The original sample code for reference:

num = 6
print("I am 6. I am afraid!")
if 5 < num:
    print("6 you are bigger than 5")
if 10 > num:
    print("I am 10. No need to be afraid of me!")
if 6 == 6:
    print("But I am 6 and 7 8 9!")

🔮 Guessing Game

Can you build a simple number guessing program? We have defined a target number, number = 5, and read the user's guess.

💡 Assignment vs Comparison: Remember, a single = is used to store values inside variables (e.g. guess = 3). To compare values, you must use a double equals sign: == (e.g. guess == 5). You can also use < (less than) and > (greater than).
🎯 Your Task

Complete the comparison operators in the three if statements to verify the user's guess:

  1. Check if guess is equal to number. If so, print: Correct!
  2. Check if guess is less than number. If so, print: Too Low!
  3. Check if guess is greater than number. If so, print: Too High!
⚙️ How Tests Work: The test runner checks three guesses:
• Inputting 5 must print Correct!.
• Inputting 3 must print Too Low!.
• Inputting 7 must print Too High!.
📋 Original Code

Fill in the gaps where the spaces are next to the comparison conditions:

number = 5
print("I have thought of a number between 1 and 10")
guess = int(input("Can you guess what it is?"))

if guess  number:
  print("Correct!")
if guess  number:
  print("Too Low!")
if guess  number:
  print("Too High!")

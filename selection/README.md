# Python Selection Statements Puzzle Course

This course is designed for Python Code Lab.

All Python files and tutorial HTML files are in one folder:

`selection`

Each Python file starts as a mini puzzle. The starter code is intentionally incomplete or incorrect, so students need to edit the code to pass the tests.

## Includes

1. **football.py** - Football Raining Check (If/Else). Determine if game can be played based on whether it is raining.
2. **numbereat.py** - Number Eat (If/Elif/Else). Evaluate number ranges and print appropriate messages.
3. **magician.py** - Magician Guess (Comparison). Validate guess against magician's number (Too High, Too Low, Correct!).
4. **letters.py** - Letter Count Check (Equality). Verify if the number of letters is 26 and print correct/wrong message.
5. **nums.py** - Larger Number Picker (Comparison). Input two numbers and output the larger one.
6. **bugs.py** - Bug Comparison (Comparison). Print which number is larger or if they are the "same".
7. **divides.py** - Divisibility Check (Modulo). Output whether a number divides by 3.
8. **clocktime.py** - 12h to 24h Clock (Selection). Convert hours and AM/PM indicators to 24h clock strings.
9. **movies.py** - Movie Ticket Pricing (If/Elif/Else). Print the total ticket price depending on the age group.

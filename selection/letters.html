How Many Letters? 🔤

🔠 The Alphabet Quiz

Do you know how many letters are in the English alphabet? Let's write a program that quizzes the user and tells them if they got it right!

🎯 Your Task

Complete the program to check the user's input:

  1. Check if the user entered 26. If it is correct, output the message: 
    "Correct Number of letters!"
  2. If they entered any other number, output the message: 
    "Wrong number of letters!"
⚙️ How Tests Work: The test runner checks two answers:
• Inputting 26 must output a message containing Correct.
• Inputting 10 must output a message containing Wrong.

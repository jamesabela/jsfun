LEVEL 1: SELECTION

Good Weather ☀️

⚽ Selection Statement

We are now looking at our first selection statement. In this case, if we type in a lower case y, it will say we can't play football!

🎯 Your Task

Your task is to change the code so that it asks if it is:

"Is it sunny?"

If it is sunny (type y), then it will say:

"Don't forget to put on sunscreen!"

And if it is not, then it will say:

"You'd best take an umbrella!"
⚙️ How Tests Work: The test runner will run your program twice. First, it will enter y and expect to see sunscreen in the output. Next, it will enter n and expect to see umbrella in the output. Make sure your prints include these keywords!
📋 Original Code

Just in case you need to start again:

test = input("Is it raining?")
if test == "y":
    print("Oh dear, no football today!")
else: 
    print("Great, let's go and play football!")

# Python Lists Course

This folder contains a one-folder Python Code Lab course for learning lists in Python.

## Purpose

Students practise creating lists, using built-in list functions, indexing, slicing, appending, removing, sorting, using related lists, and choosing random items from a list.

## Includes

1. `01_create_list.py` and `01_create_list.html` - create and print a list.
2. `02_list_stats.py` and `02_list_stats.html` - use `sum`, `min`, `max`, `len` and mean average.
3. `03_city_slice.py` and `03_city_slice.html` - use indexes and slices.
4. `04_character_list.py` and `04_character_list.html` - append user input and sort a list.
5. `05_remove_item.py` and `05_remove_item.html` - remove an item, sort, and print neatly.
6. `06_bus_ticket.py` and `06_bus_ticket.html` - use related lists for a ticketing system.
7. `07_roasting_machine.py` and `07_roasting_machine.html` - use `random.choice` with a list.
8. `08_lazy_report_writer.py` and `08_lazy_report_writer.html` - combine lists, random choice and a loop.

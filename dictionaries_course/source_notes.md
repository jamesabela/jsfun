# Source Notes

The course is based on the dictionary notes supplied by James Abela in the prompt.

The Romeo and Juliet sample is a short public domain classroom sample based on William Shakespeare's *Romeo and Juliet*. Project Gutenberg provides a public domain ebook copy in the United States and describes the ebook as reusable under the Project Gutenberg licence.

Relevant source:

- Project Gutenberg, *The Tragedy of Romeo and Juliet* by William Shakespeare: https://www.gutenberg.org/ebooks/1513

The word-frequency level uses `#load https://raw.githubusercontent.com/jamesabela/jsfun/refs/heads/main/dictionaries_course/romeo_juliet_sample.txt` so Python Code Lab can load the text file in the browser environment.

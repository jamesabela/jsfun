# Python Dictionaries Course

This course teaches dictionaries for Python Code Lab using small puzzle levels and a final creative project.

## Includes

1. `01_make_dictionary.py` - Make a dictionary
2. `02_key_rules.py` - Keys and values
3. `03_access_get.py` - Access values safely
4. `04_loop_dictionary.py` - Loop through a dictionary
5. `05_update_dictionary.py` - Add and update entries
6. `06_delete_pop.py` - Delete and pop values
7. `07_views_in.py` - Keys, values, items and in
8. `08_nested_dictionary.py` - Nested dictionaries
9. `09_word_frequency.py` - Word frequency with Romeo and Juliet
10. `10_final_project.py` - Final project: interactive dictionary app

Extra files:

- `glossary.html` - interactive dictionary vocabulary and command table
- `romeo_juliet_sample.txt` - short public domain sample for word-frequency work
- `dictionary_reference.py` - reference playground based on the original notes
- `teachers.md` - teacher notes and solutions
- `source_notes.md` - source and deployment notes

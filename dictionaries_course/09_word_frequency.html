💡 Big idea

Dictionaries are excellent for frequency tables. The word becomes the key and the count becomes the value.

This level uses a short public domain sample from Romeo and Juliet. The aim is to count how many different words appear and how many times each word appears.

⚙️ Your task

Complete the missing lines so each word is counted. This level uses #load to load the text file into Python Code Lab.

import string

word_counts = {}

    text = file.read().lower()

for mark in string.punctuation:
    text = text.replace(mark, " ")

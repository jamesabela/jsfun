Dictionary Vocabulary

Glossary
📘 Key terms

Use this page while working through the course. A dictionary stores data using key-value pairs. Unlike a list, it is looked up using keys rather than numeric indexes.

TermMeaning
DictionaryA mutable data structure that stores data as key-value pairs.
KeyThe unique identifier used to look up a value in a dictionary.
ValueThe data stored against a key. Values can be strings, numbers, lists or even other dictionaries.
Key-value pairOne complete dictionary entry, such as "France": "Paris".
ImmutableA value that cannot be changed after it is created. Strings, numbers and tuples can be dictionary keys.
MutableA value that can be changed after it is created. Lists are mutable, so they cannot be dictionary keys.
Insertion orderSince Python 3.7, dictionaries keep keys in the order they were added.
get()Returns the value for a key, or a default value if the key is missing.
keys()Returns a view of the keys in a dictionary.
values()Returns a view of the values in a dictionary.
items()Returns a view of the key-value pairs in a dictionary.
pop()Removes a specified key and returns its value.
popitem()Removes and returns the most recently inserted key-value pair.
update()Adds or changes key-value pairs using another dictionary.
Nested dictionaryA dictionary stored inside another dictionary.
Frequency tableA structure that counts how often each item appears, often built using a dictionary.

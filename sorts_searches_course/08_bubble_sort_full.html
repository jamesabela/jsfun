💡 What this means

A full bubble sort usually has an outer loop for passes and an inner loop for comparisons within each pass. In the worst case it needs repeated passes over the data, so the usual exam answer is O(n^2).

🌍 Examples
  • Sorting marks from lowest to highest.
  • Sorting small game scores.
  • Sorting classroom scores without using built-in sort.

# Sorts and Searches with Big O

This course teaches linear search, binary search, bubble sort, insertion sort and Big O notation for Cambridge 9618-style algorithm work. It includes Python starter tasks, turtle visualisations, interactive HTML algorithm animations and exam-support pages.

## Includes

1. `01_big_o_intro.py` - Why Big O Matters
2. `02_linear_search.py` - Linear Search
3. `03_linear_turtle.py` - Turtle Linear Search
4. `04_binary_search_concept.py` - Binary Search: Check the Middle
5. `05_binary_search_code.py` - Binary Search Code
6. `06_worst_case.py` - Why 9618 Uses Worst Case
7. `07_bubble_sort_concept.py` - Bubble Sort: Compare Neighbours
8. `08_bubble_sort_full.py` - Bubble Sort Full Algorithm
9. `09_bubble_turtle.py` - Turtle Bubble Sort Bars
10. `10_insertion_sort.py` - Insertion Sort
11. `11_compare_algorithms.py` - Compare Algorithms
12. `12_final_project.py` - Final Project: Algorithm Arcade

## Extra reference pages

- `vocabulary.html` - interactive glossary for algorithms and Big O
- `big_o_lab.html` - interactive Big O growth comparison
- `exam_support.html` - worst-case Big O explanations and sentence frames

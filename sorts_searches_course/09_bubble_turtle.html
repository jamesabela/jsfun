💡 What this means

This tutorial level draws the list as bars with turtle. It is not a perfect animation engine, but it helps students see that bubble sort is about repeated comparisons and swaps.

🌍 Examples
  • Height bars make the order visible.
  • Students can change the list and predict the result.
  • Turtle makes algorithm state easier to discuss.
⚙️ Your task

Run the program, then change the list values. Keep the values small enough to fit inside the -200 to 200 drawing area.

# Breadcrumb: Level 9 of 12 - Sorts and Searches

import turtle

values = [7, 3, 9, 2, 6]

def draw_bars():
    turtle.clear()
    x = -170
    turtle.penup()
    for value in values:
        turtle.goto(x, -160)
        turtle.pendown()
        for i in range(2):
            turtle.forward(40)
            turtle.left(90)
            turtle.forward(value * 20)
            turtle.left(90)
        turtle.penup()
        turtle.goto(x + 20, -180)
        turtle.write(value, align="center", font=("Arial", 12, "bold"))
        x = x + 60

turtle.speed(0)
draw_bars()

for pass_num in range(len(values)):
    for i in range(len(values) - 1):
        if values[i] > values[i + 1]:
            temp = values[i]
            values[i] = values[i + 1]
            values[i + 1] = temp
            draw_bars()

turtle.hideturtle()

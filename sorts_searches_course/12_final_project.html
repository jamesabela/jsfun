💡 What this means

The final task asks students to combine searching, sorting, Big O reasoning and visual explanation. There are no direct tests because the aim is to build and explain a working algorithm tool.

🌍 Examples
  • A revision app that lets users choose an algorithm and see the steps.
  • A turtle visualiser that draws each pass of a sort.
  • A comparison tool that counts comparisons and explains Big O.
⚙️ Your task

Create an Algorithm Arcade. It should let the user choose at least two algorithms, run them on a list, count comparisons, and print a Big O explanation for each one.

Choose two or more: linear search, binary search, bubble sort, insertion sort.
Add: comparison counter, result, Big O explanation.

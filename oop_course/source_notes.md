# Source Notes

This course was guided by the two uploaded OOP presentations.

## 01 Full Intro To OO Presentation.pdf

Useful ideas included:

- The RPG metaphor for teaching OOP.
- A `Character` class with attributes such as name, attack, defence, health and experience.
- The importance of `__init__`, `self`, attributes and methods.
- Creating instances and calling methods using dot notation.
- Inheritance through `wizard(Character)` and `knight(Character)`.
- Polymorphism through child classes using the same method name with different behaviour.
- Getter and setter examples with private health.
- Aggregation and containment as exam-relevant ideas.

## 02 Mass Creation of Classes with Lists (Student Version).pdf

Useful ideas included:

- Creating many objects using a list.
- Storing monster objects in a collection.
- The issue that directly printing an object may show a memory location.
- The need to call useful methods or define a representation.
- Building towards an RPG capstone challenge.

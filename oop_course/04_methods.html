💡 Big idea

A method is like a function, but it belongs to a class. A method is called on an object using dot notation, such as hero.print_basics().

Methods are useful because the behaviour belongs with the object. A character can print its own details, attack, take damage or gain experience.

Complete the method call so the character prints its details.

class Character:
    def __init__(self):
        self.name = "Merlin"
        self.magic = 50

    def print_basics(self):
        print("Name:", self.name)

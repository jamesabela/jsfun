Vocabulary
📝 How this appears in exams
  • Write a class with a constructor and private attributes.
  • Create an object from a class and call methods using dot notation.
  • Use a getter to return a private value.
  • Use a setter to update a private value.
  • Explain inheritance between a parent class and a child class.
  • Identify polymorphism when child classes use the same method name in different ways.
  • Use aggregation or containment where one object stores another object.
  • Create a list of objects and loop through it.
Example style question

A game stores different types of character. All characters have a name and health. A wizard also has magic. A knight also has armour. Write class definitions for this system and create one wizard and one knight.

class Character:
    def __init__(self, name, health):
        self.__name = name
        self.__health = health

    def get_health(self):
        return self.__health
Common mistakes
  • Forgetting self in the method header.
  • Forgetting to call the parent constructor inside the child constructor.
  • Trying to access a private attribute directly instead of using a getter.
  • Calling a method on the class name instead of an object name.
  • Using many repeated variables instead of a list of objects.

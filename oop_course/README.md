# Python OOP Course

This course introduces object oriented programming for Cambridge A-level Computer Science 9618 using Python and an RPG metaphor.

## Purpose

Students learn how to create classes, construct objects, store attributes, write methods, use getters and setters, apply inheritance and polymorphism, use aggregation, and store objects in lists.

This course deliberately does not cover OOP abstract data types or binary trees. Those will be taught in a separate course.

## Includes

1. `01_why_oop.py` and `01_why_oop.html` - Why objects help in bigger programs
2. `02_class_blueprint.py` and `02_class_blueprint.html` - Create a Character class
3. `03_constructor_attributes.py` and `03_constructor_attributes.html` - Constructor, self and attributes
4. `04_methods.py` and `04_methods.html` - Methods make objects do things
5. `05_instances.py` and `05_instances.html` - Many instances from one class
6. `06_getters_setters.py` and `06_getters_setters.html` - Private attributes, getters and setters
7. `07_inheritance.py` and `07_inheritance.html` - Inheritance creates specialised characters
8. `08_polymorphism.py` and `08_polymorphism.html` - Polymorphism: same method, different behaviour
9. `09_abstraction.py` and `09_abstraction.html` - Abstraction: choose the important details
10. `10_aggregation.py` and `10_aggregation.html` - Aggregation and containment
11. `11_objects_in_lists.py` and `11_objects_in_lists.html` - Create many objects using a list
12. `12_final_rpg.py` and `12_final_rpg.html` - Final project: RPG party builder
13. `vocabulary.html` - Key definitions for OOP terminology.
14. `exam_examples.html` - Common exam-style prompts and mistakes.
15. `teachers.md` - Teacher notes and solution code.
16. `syllabus_notes.md` - Course coverage notes.
17. `source_notes.md` - Notes on the source presentations used.

# Syllabus Notes - OOP

This course supports Cambridge A-level Computer Science 9618 object oriented programming concepts in Python.

Covered concepts:

- Object oriented programming
- Class
- Object
- Instance
- Attribute
- Method
- Constructor
- `__init__`
- `self`
- Getter
- Setter
- Private attribute
- Abstraction
- Inheritance
- Superclass / base class
- Subclass / derived class
- Polymorphism
- Method overriding
- Aggregation
- Containment
- Lists of objects

Not covered in this course:

- OOP abstract data types
- Binary trees
- File loading of large object datasets

These will be covered in separate courses.

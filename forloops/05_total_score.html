LEVEL 5: RUNNING TOTALS

Total Score

💡 Idea

A running total is a variable that changes each time the loop repeats.

⚙️ Task

Ask the user for three scores and print the total.

total = 0
for i in range(3):
    score = int(input("Score: "))
    total = total + score
print(total)

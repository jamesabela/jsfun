Before binary trees

Skills needed

This course is a skills culmination

Binary trees combine several hard A2 skills. Students should know these first:

  • Recursion: base case, general case and function calls that reduce the problem.
  • OOP: classes, objects, attributes, methods and references between objects.
  • File handling: loading a text file, reading each line and stripping newline characters.
  • Lists and 2D lists: using indexes to store data and pointers.

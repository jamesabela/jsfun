# Syllabus Notes

This course is designed for Cambridge International AS & A Level Computer Science 9618 A2-level binary tree work.

Students should be able to:

- Understand binary trees as an abstract data type.
- Understand that a binary tree can be implemented using arrays/lists or objects.
- Insert data into a binary search tree.
- Search a binary search tree.
- Traverse a binary tree using recursion.
- Use a null pointer value, such as `-1` or `None`.
- Read data from a file and insert the data into a suitable data structure.

The course deliberately avoids repeating the full ADT and OOP courses, but it links to them as prior knowledge.

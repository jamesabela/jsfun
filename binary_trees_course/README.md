# Binary Trees Course

This folder contains a Python Code Lab course for Cambridge International AS & A Level Computer Science 9618 binary trees.

## Purpose

This course builds binary trees as a culmination topic. It combines:

- Recursion
- OOP
- List-based data structures
- File loading
- Searching and traversal
- Cambridge-style pointer/index thinking

## Includes

1. `01_tree_idea.py` - binary tree idea and left/right placement
2. `02_list_insert.py` - insert into a list-based tree
3. `03_list_search.py` - search a list-based tree
4. `04_recursive_traversal.py` - recursive in-order traversal
5. `05_turtle_tree.py` - visual tree drawing with turtle
6. `06_oop_node.py` - Node class with left and right references
7. `07_oop_insert.py` - recursive OOP insertion
8. `08_oop_search.py` - recursive OOP search
9. `09_load_words.py` - load words from a text file
10. `10_file_to_list_tree.py` - load a text file into a list-based binary tree
11. `11_file_to_oop_tree.py` - load a text file into an OOP binary tree
12. `12_final_project.py` - final Binary Tree Word Explorer project

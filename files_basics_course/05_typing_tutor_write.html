Typing Tutor: Save Latest WPM

Why files matter

A typing tutor can measure how fast the user typed. This version keeps it simple: the user types one whole line and the program saves the latest WPM.

Before: memory disappears
After: file keeps data
Your task

Store the latest WPM in latest_wpm.txt. wpm is a number, so convert it to a string before writing.

file = open("latest_wpm.txt", "w")
file.write(str(wpm))
file.close()
Helpful hint
Files store text. Convert numbers using str() before writing them.

FINAL PROJECT: FILES BASICS

Typing Champion 🏆

💡 Scenario

A school coding club wants a simple typing game. Players type one full line as accurately and quickly as possible. The program should save the latest speed so it can still be seen after the program has closed.

This final project is not input/output assessed. Your teacher will check that the program works and that your code is clear.
⚙️ Requirements
  • Use a list of typing phrases.
  • Choose one phrase for the player to type.
  • Measure how long the player takes.
  • Calculate the words per minute.
  • Save the latest WPM as one line in latest_wpm.txt.
  • Read the old high score from highscore.txt.
  • If the latest WPM is higher than the old high score, save the new high score as one line.
  • Use suitable output messages so the player knows what happened.
  • Add comments to explain the important parts of your program.

# Source Notes - Recursion Course

This course was built for Cambridge International AS & A Level Computer Science 9618.

Useful sources used when designing the course:

- MIT OpenCourseWare 6.100L Lecture 15: Recursion. MIT describes recursion as a programming method and a way to divide and conquer, using a base case and recursive step.
  https://ocw.mit.edu/courses/6-100l-introduction-to-cs-and-programming-using-python-fall-2022/pages/lecture-15-recursion/

- MIT OpenCourseWare 6.0001 Lecture 6: Recursion and Dictionaries. This shows recursion as part of MIT's introductory Python course.
  https://ocw.mit.edu/courses/6-0001-introduction-to-computer-science-and-programming-in-python-fall-2016/resources/lecture-6-recursion-and-dictionaries/

- Cambridge International AS & A Level Computer Science 9618 syllabus. Recursion includes writing and tracing recursive algorithms, when recursion is beneficial, and compiler awareness.
  https://www.cambridgeinternational.org/Images/721397-2027-2029-syllabus.pdf

- Cambridge 9618 Paper 4 May/June 2024 included recursive binary search in one variant.
  https://dynamicpapers.com/wp-content/uploads/2015/09/9618_s24_qp_42.pdf

- Cambridge 9618 Paper 4 Oct/Nov 2021 included recursive binary tree traversal in one variant.
  https://pastpapers.co/caie/a-level/computer-science-%28for-first-examination-in-2021%29-%289618%29/2021-oct-nov/9618_w21_qp_41.pdf

- Teacher-provided recursion slides and compiler notes were also used to shape the sequence, especially base case, general case, recursive call, stack frames and unwinding.

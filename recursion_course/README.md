# Python Recursion Course

This folder contains a Python Code Lab course on recursion for Cambridge International AS & A Level Computer Science 9618.

The course emphasises recursion as a mathematical and visual idea. It includes turtle examples, interactive HTML explanations, exam links, and creative project work.

## Includes

1. `01_what_is_recursion.py` - basic countdown recursion.
2. `02_base_case_turtle_circle.py` - turtle circles and the importance of a base case.
3. `03_factorial_math.py` - factorial as a mathematical recursive definition.
4. `04_call_stack_unwinding.py` - stack frames and unwinding.
5. `05_fibonacci_sequence.py` - Fibonacci recursion and repeated work.
6. `06_gcd_euclid.py` - Euclid's recursive greatest common divisor algorithm.
7. `07_recursive_cinema.py` - drawing cinema rows using recursive turtle calls.
8. `08_recursive_binary_search.py` - recursive binary search for 9618 Paper 4 style practice.
9. `09_towers_of_hanoi.py` - classic recursive problem-solving.
10. `10_gray_code.py` - recursive pattern generation.
11. `11_turtle_tree.py` - recursive fractal tree using turtle.
12. `12_final_project.py` - creative fractal cinema project.

## Reference files

- `vocabulary.html` - key recursion terms.
- `exam_examples.html` - 9618 exam links and answer frames.
- `source_notes.md` - sources used when designing the course.
- `teachers.md` - teacher guide and solution key.

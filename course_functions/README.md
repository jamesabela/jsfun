# Python Procedures and Functions

This folder contains a single-folder Python Code Lab course on procedures and functions for Cambridge International AS & A Level Computer Science 9618.

## Purpose

Students learn how to define and use procedures and functions in Python, while also learning the Cambridge 9618 vocabulary used in pseudocode questions.

## Breadcrumbs

Each tutorial page and Python starter file includes a simple progress breadcrumb, for example:

`Level 4 of 10 - First Function Return`

This is not intended as navigation. It simply tells students how far they are through the course.

## Includes

1. `01_first_procedure.py` - define and call a simple procedure.
2. `02_parameters.py` - use parameters and arguments.
3. `03_turtle_square.py` - create a reusable turtle procedure.
4. `04_function_return.py` - return a value from a function.
5. `05_use_function_expression.py` - use a function call inside a calculation.
6. `06_scope.py` - local variables, global variables and scope.
7. `07_early_return.py` - use return for validation and early exits.
8. `08_value_reference.py` - explore BYVAL, BYREF and Python list mutation.
9. `09_decomposition.py` - break a ticket machine into subroutines.
10. `10_final_project.py` - combine procedures, functions and decomposition.

Each Python file has a matching tutorial page with the same number and name.

## Notes on testing

This course uses fewer automated tests than some earlier courses. Procedures and functions often need students to read, edit, run, explain and improve code rather than only match a fixed output. The tests are deliberately light-touch so that students can progress while still being encouraged to experiment.

By Value and By Reference Ideas 📦

Level 8 of 10
💡 Idea

Cambridge pseudocode uses BYVAL and BYREF. BYVAL means a copy is passed. BYREF means the original variable can be changed by the procedure.

passing by valuepassing by referencemutable dataprocedure
🎓 Cambridge 9618 link

Cambridge says that if no method is specified, passing by value is assumed. Python does not use BYVAL or BYREF keywords, but lists are mutable, so a procedure can change the contents of a list.

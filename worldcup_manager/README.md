# World Cup Dream Manager

A GCSE-level Python course where students build a football manager program. Students can choose a women's or men's team and take any country towards a fictional World Cup dream.

The course uses only modules supported by Python Code Lab: `random`, `math`, `statistics`, `datetime`, `string`, `turtle` and `time`. This course mainly uses `random` and `turtle`.

## Includes

1. `01_choose_your_dream.py` - input, output and variables
2. `02_team_type.py` - selection and `.lower()`
3. `03_country_rating.py` - integer input and simple maths
4. `04_pick_a_squad.py` - lists and appending
5. `05_training_points.py` - integer division and rating boosts
6. `06_random_match.py` - random match simulation
7. `07_goal_difference.py` - points and goal difference
8. `08_group_table.py` - simple 2D lists
9. `09_penalty_shootout.py` - loops and random outcomes
10. `10_turtle_stadium.py` - creative turtle drawing
11. `11_load_country_data.py` - loading CSV-style text data
12. `12_build_your_world_cup.py` - final combined project

## Data files

- `countries.csv` - country ratings and stories
- `players.csv` - fictional players for men and women
- `fixtures.csv` - group stage fixture examples

The player names are fictional. The country names are real and the ratings are classroom-friendly values inspired by the idea of football ranking points rather than official live data.

Half-life is the time taken for the activity, count rate, or number of undecayed nuclei in a radioactive sample to fall to half its original value.

Teacher notes

This tool uses repeated halving for whole-number half-lives because that is the most useful iGCSE method. It also uses the logarithm formula in the second calculator so it can handle answers that are not exact whole numbers.

# Python Error Detectives: Top 10 Beginner Mistakes

A 10-level Python Code Lab course for absolute beginners. Students repair one common error in each program and use automated tests to confirm the fix.

## Course order

1. `01_capitalisation.py` - Capitalisation: Case Closed?
2. `02_speech_marks.py` - Speech Marks: The Missing Quotes
3. `03_brackets.py` - Brackets: Close the Case
4. `04_indentation.py` - Indentation: Line Up the Evidence
5. `05_colon.py` - Colons: The Missing Signal
6. `06_variable_order.py` - Variable Order: Create Before Use
7. `07_variable_spelling.py` - Variable Names: A Suspicious Spelling
8. `08_strings_numbers.py` - Data Types: Text or Number?
9. `09_equals.py` - Equals Signs: Assign or Compare?
10. `10_infinite_loop.py` - Infinite Loops: Escape the Loop

## Teaching approach

Ask students to run each broken program before editing it. They should read the error message, identify the line, make one small repair and run the tests. Level 10 is a logic error, so the program runs but does not stop until the counter is updated.

# Syllabus Notes - Cambridge 9618 OOP ADTs

This course supports the A2 abstract data type material using Python classes.

Key 9618 links:

- An ADT is a collection of data and operations on that data.
- Stack, queue and linked list are examples of ADTs.
- Students should know how to add, edit and delete data from these structures.
- At A2, students should be able to write algorithms to insert into stack, queue and linked list, delete from stack, queue and linked list, and find an item in a linked list.
- Students should understand that an ADT can be implemented using built-in types or another ADT.
- Binary trees are deliberately excluded from this course and should be taught separately.

Course design choices:

- Python classes are used to model each ADT.
- Private attributes are used to encourage encapsulation.
- Lists are used internally for stack and queue examples.
- Linked lists use `Node` objects and `next` references.
- Interactive HTML animations are included to support tracing and pointer movement.

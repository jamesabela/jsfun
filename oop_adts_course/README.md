# OOP Abstract Data Types Course

This course teaches abstract data types for Cambridge International AS & A Level Computer Science 9618 using object-oriented Python.

It covers stacks, queues and linked lists as classes. Binary trees are deliberately left out for a later course.

## Includes

1. `01_adt_interface.py` - ADT, interface and implementation.
2. `02_stack_class.py` - Stack class constructor and private attributes.
3. `03_stack_push.py` - Push and overflow.
4. `04_stack_pop_peek.py` - Pop, peek and underflow.
5. `05_queue_class.py` - Queue class using list storage.
6. `06_circular_queue.py` - Circular queue and pointer wraparound.
7. `07_queue_application.py` - Printer queue application.
8. `08_node_class.py` - Node class for linked lists.
9. `09_linked_list_insert_traverse.py` - Insert and traverse.
10. `10_linked_list_search_delete.py` - Search and delete idea.
11. `11_adt_from_adt.py` - Queue made from two stacks.
12. `12_final_project.py` - RPG control desk project.

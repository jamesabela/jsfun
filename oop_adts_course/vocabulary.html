📚 Terms
Abstract data type (ADT)

A collection of data and a set of operations on that data. The user focuses on what the ADT does, not how it is stored internally.

Interface

The public operations provided by an ADT, such as push, pop, enqueue or dequeue.

Implementation

The hidden code and storage used to make the ADT work.

Encapsulation

Keeping data and methods together inside a class, often hiding internal data using private attributes.

Stack

A Last In, First Out ADT. The most recently added item is removed first.

Push

Add an item to the top of a stack.

Pop

Remove and return the item from the top of a stack.

Peek

Look at the top item of a stack without removing it.

Overflow

An error caused by trying to add an item to a full fixed-size structure.

Underflow

An error caused by trying to remove an item from an empty structure.

Queue

A First In, First Out ADT. The first item added is the first item removed.

Enqueue

Add an item to the rear of a queue.

Dequeue

Remove and return the item from the front of a queue.

Front pointer

A pointer or index showing where the next queue item will be removed from.

Rear pointer

A pointer or index showing where the newest queue item was added.

Circular queue

A queue that reuses storage by wrapping the rear pointer around to the start.

Linked list

An ADT made from nodes where each node stores data and a link to the next node.

Node

One item in a linked list. It usually stores data and a next reference.

Head or start pointer

A reference to the first node in a linked list.

Null pointer

A marker showing that there is no next node. In Python this is usually None.

Traversal

Visiting each node in a linked list by following next references.

Class

A blueprint for creating objects. In this course, each ADT is implemented as a class.

Object

An instance of a class, such as one particular Stack or Queue object.

Constructor

A special method used to create and initialise an object. In Python this is __init__.

Private attribute

An attribute intended to be accessed only through class methods, often written with double underscores in Python.

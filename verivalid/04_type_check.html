Type Check

Level 4 of 10
💡 What this means

A type check makes sure data is the correct type, such as integer, real, string or Boolean. In Python, input starts as a string, so programs often check before converting it.

🌍 Real-life examples
  • The number of tickets should be an integer, not text such as four.
  • A price can be a real number such as 12.50.
  • A name is stored as a string.
  • A yes/no setting could be stored as a Boolean value.

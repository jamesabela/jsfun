# Python Validation and Verification Course

Folder: `verivalid`

This course introduces validation and verification for iGCSE Computer Science style programming in Python. It covers the need for validation checks on input data, the main types of validation check, and the two main verification checks.

## Course progress breadcrumbs

Each tutorial page and Python file shows a progress marker such as `Level 3 of 10`. These breadcrumbs are not navigation links. They are there so students know how far they are through the course.

## Includes

1. `01_validate_or_verify.py` and `01_validate_or_verify.html` - difference between validation and verification
2. `02_range_check.py` and `02_range_check.html` - range check
3. `03_length_check.py` and `03_length_check.html` - length check
4. `04_type_check.py` and `04_type_check.html` - type check
5. `05_presence_check.py` and `05_presence_check.html` - presence check
6. `06_format_check.py` and `06_format_check.html` - format check
7. `07_check_digit.py` and `07_check_digit.html` - check digit
8. `08_visual_check.py` and `08_visual_check.html` - visual check verification
9. `09_double_entry.py` and `09_double_entry.html` - double entry verification
10. `10_combined_form.py` and `10_combined_form.html` - final combined validation and verification challenge

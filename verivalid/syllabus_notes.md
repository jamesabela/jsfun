# Syllabus Notes - Validation and Verification

This course is based on iGCSE Computer Science 0478 validation and verification ideas.

## Validation

Validation checks whether input data is reasonable and acceptable before it is processed. Validation does not prove that the data is correct. It only checks whether it follows the rules set by the system.

Covered checks:

- range check
- length check
- type check
- presence check
- format check
- check digit

## Verification

Verification checks whether data has been copied accurately from another source. Verification does not prove that the original data was correct.

Covered checks:

- visual check
- double entry check

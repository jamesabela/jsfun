# Python Operators Puzzle Course

This course is designed for Python Code Lab.

All Python files and tutorial HTML files are in one folder:

`operators`

Each Python file starts as a mini puzzle. The starter code is intentionally incomplete or incorrect, so students need to edit the code to pass the tests.

## Includes

1. **start.py** - Lesson 1: Snack Shop Secret Code (Addition). Calculate total code by adding snack and drink prices.
2. **02_robot_battery.py** - Lesson 2: Robot Battery Rescue (Subtraction). Work out battery remaining after travel.
3. **03_space_fuel.py** - Lesson 3: Space Fuel Calculator (Multiplication). Calculate total fuel needed for space jumps.
4. **04_pizza_party.py** - Lesson 4: Pizza Slice Share (Division). Use normal float division to share slices.
5. **05_pirate_treasure.py** - Lesson 5: Pirate Treasure Split (Floor Division). Calculate equal share of whole coins.
6. **06_leftover_loot.py** - Lesson 6: Leftover Loot (Modulo). Find remaining coins after equal distribution.
7. **07_power_up.py** - Lesson 7: Arcade Power-Up (Exponentiation). Raise score to the power of the power-up.

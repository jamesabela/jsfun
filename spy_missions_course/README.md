# Spy Missions: Python Codebreakers

A connected spy-themed Python Code Lab course built from the uploaded KLCC mission files.

## Course order

1. `mission_01.py` - The Steel Door: lists and slicing.
2. `mission_02.py` - The Factor Lock: factors and loops.
3. `mission_03.py` - The Identity Console: arithmetic sequences.
4. `mission_04.py` - Mainframe Countdown: sequence formulae.
5. `mission_05.py` - Encrypted Symbols: strings and counting.
6. `mission_06.py` - The LCM Cipher: gcd and lcm.
7. `mission_07.py` - Escape Route Cleanup: lists and duplicates.
8. `mission_08.py` - Factorial File: for loops and multiplication.
9. `mission_09.py` - Reverse Factorial: while loops.
10. `mission_10.py` - Longest Word Warning: string splitting.
11. `mission_11.py` - Find The Mole: sets and median.
12. `mission_12.py` - Whale Number Door: digit algorithms.
13. `mission_13.py` - Increasing Trail: subarrays.
14. `mission_14.py` - Bomb Password Words: functions and place value.
15. `mission_15.py` - Largest Running Sum: Kadane algorithm.
16. `mission_16.py` - Palindrome Maker ID: nested loops and strings.
17. `mission_17.py` - Numopolis Artefact: dynamic programming.
18. `mission_18.py` - Ledger Caesar Trail: Caesar cipher.
19. `mission_19.py` - Traitor Damage Scan: sliding window.
20. `mission_20.py` - The Hidden Passage: recurrence and modulo.

## Notes

- Arrays/lists are entered as space-separated values so that the automated test parser does not split them incorrectly at commas.
- Some original extension tasks mentioned missing external files or images. These have been adapted into self-contained tests.
- The Gold missions are harder and work well as extension tasks.

Ledger Caesar Trail

Spy Missions - Caesar cipher

Story

A ledger contains words shifted by different Caesar keys. This training version gives the keys so agents can practise decoding.

Your coding task

Input format: encoded words|shifts. Decode each word using the matching shift.

Example input

mjqqt dvysk|5 7

Expected output

hello world

Agent hint

Solve the small example first, then run the tests. Keep the output simple: print only the answer unless the task asks for extra text.

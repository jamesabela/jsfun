File Handling in Python

Learn how to load, read, and write files in the Python Code Lab browser editor.

How it works in Python Code Lab

In a browser sandbox, Python runs in a virtual memory space. To load external files, you can use the special #load URL directive at the top of your code. This downloads the file directly into Python's virtual memory so your code can read it using standard file operations!

Take a look at the three main components of the starter code:

  1. The with statement: This opens the file and automatically closes it when the block finishes. Always use this format, as it is much safer than manual open() and close() calls.
  2. Reading with loops: We can loop through whole_file directly using a for loop. Python automatically knows the length of the file, so you do not have to worry about looking for the EOF (End of File).
  3. Printing: We print each line with end="" because lines inside text files already end with a newline character.

Writing vs. Appending Files

When you want Python to output data to a file, you must specify the correct mode. Unlike reading ("r"), outputting has two distinct modes:

  • Write Mode ("w"): Overwrite or Create Fresh

    Use "w" when you want to create a new file or completely replace the content of an existing file. If the file already exists, all its previous text will be deleted. 

    with open("my_file.txt", "w") as f:
    f.write("This is a fresh line of text.\n")
    f.write("This overwrites any old data in my_file.txt.\n")
  • Append Mode ("a"): Add to the End

    Use "a" when you want to add new content to the very bottom of the file while preserving all original text. This is perfect for logs or running lists. 

    with open("my_file.txt", "a") as f:
    f.write("This new line gets added at the very bottom.\n")

💡 Top Tip: The f.write() function does not automatically add a new line like print() does. Remember to add \n at the end of your text strings to start a new line!

Easy

1. Use a Variable for Filename

Change the code to store the filename in a variable, then open the file using that variable.

🔍 Show Solution

Declare your filename string first, then pass it as the first argument to open():

my_filename = "rudyard.txt"

with open(my_filename, "r") as whole_file:
    for line in whole_file:
        print(line, end="")
Medium

2. Exercise: Line Filter

Create a program that counts the total number of lines in the poem, and only prints the lines that start with the word "If".

🔍 Show Solution

Use a counter variable inside the loop, and filter lines using .strip().startswith("If"):

line_count = 0

with open("rudyard.txt", "r") as whole_file:
    for line in whole_file:
        if line.strip().startswith("If"):
            print(line, end="")
            line_count += 1

print("\n--- Summary ---")
print(f"Found {line_count} matching lines.")
Hard

3. Count Words & Save

Modify the script to count how many times the word "If" appears anywhere in the poem, and save the result to a new file named stats.txt.

🔍 Show Solution

Read the whole file contents using .read(), count the target string, and write to a new file in write mode:

with open("rudyard.txt", "r") as whole_file:
    poem_text = whole_file.read()

count_if = poem_text.count("If")
print(f"Counting done: {count_if} times.")

# Save results
with open("stats.txt", "w") as out:
    out.write(f"The word 'If' appears {count_if} times.\n")

print("stats.txt has been downloaded to your computer!")

Virtual Pet Challenges

Improve your Python virtual pet by adding new features.

Your starting point

Your virtual pet already has values such as hunger, happiness and energy. Each action should change these values.

Choose one challenge at a time. Test your program after each change.

Easy

1. Add an Age

Add a variable called age.

Each time the player chooses an action, increase the age by 1.

age = age + 1
Easy

2. Rename the Pet

Add a menu option that lets the player rename their pet.

Use input() to ask for the new name.

Easy

3. Add a Cleanliness Stat

Add a new variable called cleanliness.

Playing should make the pet less clean. Washing should make it cleaner.

Easy

4. Add Status Messages

Print different messages depending on the pet's stats.

if hunger > 7:
    print("Your pet is very hungry!")
Medium

5. Add Random Events

Use the random module to make unexpected things happen.

For example, the pet might find a toy or get muddy.

import random

event = random.randint(1, 3)
Medium

6. Add a Shop

Add coins to the game.

The player can spend coins on food, toys or medicine.

Medium

7. Add a Score

Create a score based on how healthy and happy the pet is.

score = happiness + energy + (10 - hunger)
Medium

8. Add Health

Add a health variable.

If hunger is too high or energy is too low, health should decrease.

Hard

9. Save the Pet

Save the pet's name and stats to a text file.

When the program starts, load the saved data.

Hard

10. Use Functions

Move repeated code into functions.

def show_stats():
    print("Hunger:", hunger)
    print("Energy:", energy)
Hard

11. Use a Class

Create a Pet class.

The pet's name, hunger, happiness and energy can become attributes.

class Pet:
    def __init__(self, name):
        self.name = name
        self.hunger = 5
Hard

12. Add Different Pet Types

Let the player choose a cat, dog, dragon or robot.

Each pet type could have different starting stats.

Challenge: Can you keep your pet happy, healthy and alive?

# Python Turtle Graphics Course

A ten-level beginner course for Python Code Lab.

## Upload location

Upload this complete folder to:

`jsfun/turtle_course/`

The folder name must remain exactly:

`turtle_course`

The Python files and HTML files are stored together in this repository folder, but they are loaded from different web locations:

- Tutorial HTML: `https://jamesabela.github.io/jsfun/turtle_course/...`
- Runnable Python: `https://raw.githubusercontent.com/jamesabela/jsfun/refs/heads/main/turtle_course/...`

This follows the Python Code Lab course format: the first web link in each Python file opens the tutorial preview, while `#Next` points to the next raw Python file. The first link rule and raw `#Next` format are required by the course guide.

## Course sequence

1. `01_first_steps.py` - movement and turning
2. `02_square.py` - loops and squares
3. `03_polygons.py` - variables and polygon angles
4. `04_colour_and_fill.py` - pen colour and filled shapes
5. `05_pen_control.py` - penup, pendown and goto
6. `06_circles.py` - circles and rosette patterns
7. `07_procedures.py` - procedures with parameters
8. `08_spiral.py` - changing values inside a loop
9. `09_skyline.py` - coordinates, lists and procedures
10. `10_turtle_postcard.py` - open-ended final project

## Important assessment note

The automated test checks the completion message so that course progression works. It cannot check the Turtle canvas. The tutorial therefore tells students to inspect their picture, and teachers should visually check the drawing where appropriate.

## Supported Turtle features used

- `forward()`, `right()` and `left()`
- `goto()`
- `penup()` and `pendown()`
- `pensize()` and `pencolor()`
- `fillcolor()`, `begin_fill()` and `end_fill()`
- `circle()`
- `shape()`
- `bgcolor()`

## Visual reference images

The `images/` folder contains one visual result for each level. Each HTML tutorial displays its matching image in a **Visual check** section so students can compare the structure of their Turtle drawing with the intended result.

The final project image is labelled as one possible example because students are encouraged to create an original postcard.
